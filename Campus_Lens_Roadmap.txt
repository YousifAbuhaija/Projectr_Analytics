1.2 Adapter Build Order (dependencies flow left-to-right)


Scorecard (lat/lon, city) → IPEDS (enrollment) → Census BPS (permits)
                          → ACS (housing units) → ApartmentList (rent)
                                                          ↓
                                              Housing Pressure Score
    1    College Scorecard — first because it gives us unitid, lat, lon, city, state, enrollment. Everything else keys off this.
    2    IPEDS (Urban Institute) — uses unitid from Scorecard. Returns 10yr enrollment trend.
    3    Census BPS — uses FIPS place code (derived from city/state). Returns permit counts.
    4    Census ACS — uses lat/lon to find nearby tracts. Returns housing unit count.
    5    ApartmentList / HUD — uses city name. Returns median rent + trend.
1.3 Scoring Engine
The Housing Pressure Score from the report has 3 components:

Final score = weighted average, clamped to 0–100. Higher = more pressure = better opportunity.
